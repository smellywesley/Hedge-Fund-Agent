# Report Templates

All reports are educational research artifacts. Every report ends with a
data-sources-and-freshness table and the simulation disclaimer.

## Report types

| Type | Length | Use |
|---|---:|---|
| Flash Note | 1 page | Quick review after news/earnings |
| Investment Memo | 5–10 pages | Serious research |
| Initiating Coverage | 25–40 pages | Full report |
| Portfolio Health | weekly | Paper fund review |
| Watchlist Review | periodic | Tracked-name summary |
| Risk Report | periodic | Exposure and rule status |

## Full report structure (Initiating Coverage)

1. Executive Summary
2. Company Overview
3. Business Model
4. Industry Overview
5. Competitive Positioning
6. Historical Financials
7. Key Performance Indicators
8. Management and Strategy
9. Growth Drivers
10. Catalyst Calendar
11. Valuation (assumptions + bear/base/bull + sensitivity)
12. Bull Case
13. Bear Case
14. Risks
15. Scenario Analysis
16. Investment Committee Questions
17. Red-Team Critique
18. Data Sources and Freshness
19. Appendix

Flash Note = 1, 10, 11 (summary), 12/13 (condensed), 18.
Investment Memo = all sections, condensed.

## Export formats

Markdown first (Phase 6), then HTML and PDF. DOCX/PPTX later. A report is
only exportable after the Audit Agent passes it (see agent_contracts.md).
